# CRITICS ARCHIVE

**Swag Is Art**

Minimal luxury streetwear brand website built with HTML, CSS, and JavaScript.

---

## 🖤 Brand Concept

Nothing here is accidental.
Everything is archived.

Critics Archive is a fashion brand built on documentation, not trends. Each garment functions as a cultural record — preserving tension, identity, and critique through design.

---

## 🚀 Project Overview

This website includes:

* Minimal luxury dark aesthetic
* Hero landing section
* Product grid
* Individual product view
* Animated scroll effects
* Hover interactions
* Slide-in cart drawer
* Basic cart logic (JavaScript)
* Shopify Liquid placeholders for theme compatibility

## 💻 How to Run Locally

1. Download or copy the `index.html` file.
2. Open it in a browser.

OR

Use VS Code + Live Server:

* Install the **Live Server** extension
* Right-click `index.html`
* Click **Open with Live Server**

---

## 🛒 Cart System

The cart is handled via front-end JavaScript:

* Items are stored in a temporary array
* Total updates dynamically
* Slide-in drawer animation

Note: This is not connected to a payment gateway yet.
